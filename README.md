# Personal Finance Management Application

## Overview
This is a command-line application to manage personal finances. It allows users to track their income and expenses, generate financial reports, set monthly budgets, and store data securely.

## Features
- User Registration and Authentication
- Income and Expense Tracking
- Monthly and Yearly Financial Reports
- Budget Management
- Data Persistence using SQLite3
- Backup and Restore functionality
